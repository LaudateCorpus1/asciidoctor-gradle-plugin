              • File getToolingWorkDir()

                The suggested working directory that the implementation tool should use. For instance if the implementation tool is Node.js with NPM, the directory will usually be the same for all tasks, but in cases where different versions of NPM packages are used by tasks then the working directory will be different for the specific task.

                Returns:
                Suggested working directory.
