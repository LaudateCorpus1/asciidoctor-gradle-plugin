            • @TaskAction void copy()

            • @Input String getBookPath()

              The relative path to the book directory in the Dropbox folder. This is typically the relative path to the parent of the manuscript folder.

              Returns:
              A relative path. For instance Leanpub/idiomaticgradle.

            • @OutputDirectory File getDestinationDir()

              Directory where content will be copied to.

              Returns:
              Final output directory

            • @Internal File getDropboxRoot()

              Root directory on filesystem where Dropbox synchronises all files.

              Returns:
              Path to Dropbox root on local filesystem.

            • @InputDirectory File getSourceDir()

              Directory where source files will be read from. This is the directory where converted Leanpub files are written to. it is typically named manuscript.

              Returns:
              Directory containing generated Leanpub Markuva or Markdown content.
